# TapTap - 랜딩 페이지 (SNS Changer)

[Figma Make 디자인](https://www.figma.com/make/Xs2XxhrSPwFCGXtxbvtCdf/Make-Design-Responsive)을 기반으로 한 랜딩 페이지 프로젝트입니다.  
디자인의 폰트, 색상, border-radius, 그림자 등 시각 스타일을 그대로 유지했습니다.

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 http://localhost:5173 으로 접속합니다.

## 빌드

```bash
npm run build
npm run preview
```

## 히어로 섹션 이미지

디자인에 사용된 이미지(`749fb7e3259694ae57dd4fb396c11fb9c0c96796.png`)를 그대로 쓰려면:

1. Figma Make 파일에서 해당 이미지 노드를 선택한 뒤 **Export** 로 PNG를 내보냅니다.
2. 내보낸 파일 이름을 `749fb7e3259694ae57dd4fb396c11fb9c0c96796.png` 로 맞춘 다음
3. 프로젝트의 **`public`** 폴더에 넣어 주세요.

이미지가 없어도 레이아웃과 나머지 UI는 정상적으로 표시됩니다.
